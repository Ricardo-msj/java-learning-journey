# ☕ Java Learning Journey

Este repositório documenta minha jornada prática de aprendizado em **Java**, com exercícios, anotações, desafios e mini projetos desenvolvidos ao longo dos estudos.

O objetivo é consolidar fundamentos de programação e evoluir progressivamente para desenvolvimento backend com Java.

---

## 🎯 Objetivo

Construir uma base sólida em:

- Java Fundamentals
- Programação Orientada a Objetos (POO)
- Estruturas de Dados
- Resolução de Problemas
- Banco de Dados
- Backend com Spring Boot

---
